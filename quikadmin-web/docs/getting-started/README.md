## Quick Start

If you're already familiar with React development, here's a quick overview:

```bash
# Install dependencies
bun install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Start development server
bun run dev

# Run tests
bun run test

# Build for production
bun run build
```
