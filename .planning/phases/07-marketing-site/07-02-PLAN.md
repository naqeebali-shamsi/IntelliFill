<task type="auto">
  <name>Task 1: Initialize Astro project with Tailwind</name>
  <files>marketing/package.json, marketing/astro.config.mjs, marketing/tailwind.config.mjs, marketing/tsconfig.json</files>
  <action>
    Create new Astro project in `marketing/` directory:

    1. Initialize Astro project:
       ```bash
       cd N:/Intellifill
       npm create astro@latest marketing -- --template minimal --typescript strict --no-git --no-install
       cd marketing
       npm install
       ```

    2. Add Tailwind CSS integration:
       ```bash
       npx astro add tailwind --yes
       ```

    3. Configure astro.config.mjs:
       ```javascript
       import { defineConfig } from 'astro/config';
       import tailwind from '@astrojs/tailwind';

       export default defineConfig({
         site: 'https://intellifill.com',
         integrations: [tailwind()],
         output: 'static',
       });
       ```

    4. Configure tailwind.config.mjs to match app branding:
       - Copy color palette from quikadmin-web (primary, surface colors)
       - Use same font family (Inter for body, plus heading font)
       - Keep consistent spacing and border-radius

    5. Create base Layout.astro in src/layouts/:
       - HTML boilerplate with proper meta tags
       - Open Graph meta tags for social sharing
       - Favicon and apple-touch-icon references
       - Google Fonts link for Inter

    6. Verify setup:
       ```bash
       npm run dev
       ```
       Site should start at localhost:4321
  </action>

<task type="auto">
  <name>Task 3: Configure SEO and generate static assets</name>
  <files>marketing/src/layouts/Layout.astro, marketing/public/robots.txt, marketing/public/sitemap.xml</files>
  <action>
    Add SEO optimization:

    1. **Update Layout.astro** with comprehensive meta tags:
       ```astro
       ---
       interface Props {
         title: string;
         description?: string;
         image?: string;
       }

       const {
         title,
         description = "AI-powered document processing and form automation for UAE PRO agencies. Upload documents, extract data automatically, fill forms in seconds.",
         image = "/og-image.png"
       } = Astro.props;

       const canonicalURL = new URL(Astro.url.pathname, Astro.site);
       ---

       <!DOCTYPE html>
       <html lang="en">
       <head>
         <meta charset="UTF-8" />
         <meta name="viewport" content="width=device-width, initial-scale=1.0" />
         <title>{title}</title>
         <meta name="description" content={description} />
         <link rel="canonical" href={canonicalURL} />

         <!-- Open Graph -->
         <meta property="og:type" content="website" />
         <meta property="og:url" content={canonicalURL} />
         <meta property="og:title" content={title} />
         <meta property="og:description" content={description} />
         <meta property="og:image" content={new URL(image, Astro.site)} />

         <!-- Twitter -->
         <meta name="twitter:card" content="summary_large_image" />
         <meta name="twitter:title" content={title} />
         <meta name="twitter:description" content={description} />
         <meta name="twitter:image" content={new URL(image, Astro.site)} />

         <!-- Favicon -->
         <link rel="icon" type="image/svg+xml" href="/favicon.svg" />
         <link rel="apple-touch-icon" href="/apple-touch-icon.png" />

         <!-- Fonts -->
         <link rel="preconnect" href="https://fonts.googleapis.com" />
         <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
         <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet" />
       </head>
       <body class="bg-slate-900 text-white font-sans antialiased">
         <slot />
       </body>
       </html>
       ```

    2. **Create robots.txt**:
       ```
       User-agent: *
       Allow: /

       Sitemap: https://intellifill.com/sitemap.xml
       ```

    3. **Create sitemap.xml** (or use @astrojs/sitemap):
       ```bash
       npx astro add sitemap --yes
       ```
       This auto-generates sitemap on build.

    4. **Create placeholder OG image**:
       - Create a simple 1200x630 PNG with logo and tagline
       - Save to public/og-image.png
       - Can use a simple design tool or placeholder for now

    5. **Build and verify**:
       ```bash
       npm run build
       ```
       Check dist/ folder for generated files.
  </action>
