## Prompt 1: Pitch Deck (10 Slides)

Copy this entire prompt into Gemini:

```
Create a professional 10-slide investor/sales pitch deck for IntelliFill, an AI-powered document processing and form automation platform.

**Company Overview:**
- Product: IntelliFill - AI that automatically fills PDF forms from uploaded documents
- Target Market: UAE PRO (Public Relations Officer) agencies handling visa/company formation
- Stage: MVP ready, seeking first paying customers
- Key Metrics: 93% OCR accuracy, <2 second processing time, open-source OCR (no cloud API costs)

**Slide Structure Required:**

SLIDE 1: TITLE
- Company name: IntelliFill
- Tagline: "AI-Powered Form Automation for PRO Agencies"
- Subtitle: Stop filling forms manually. Let AI do it.

SLIDE 2: THE PROBLEM
- PRO agents spend 4+ hours daily on manual data entry
- Copy-pasting from passports, Emirates IDs, trade licenses into government forms
- Error-prone, tedious, doesn't scale
- Each client requires filling 10-20 different forms with the same data

SLIDE 3: THE SOLUTION
- Upload client documents once
- AI extracts all relevant data automatically
- Click to auto-fill any form instantly
- Visual: Show the 3-step flow (Upload → Extract → Fill)

SLIDE 4: PRODUCT DEMO
- Show screenshot/mockup of the dashboard
- Highlight key features:
  - Client management
  - Document upload with OCR
  - One-click form filling
  - Form history and templates

SLIDE 5: HOW IT WORKS
- Step 1: Upload documents (passport, Emirates ID, trade license)
- Step 2: AI extracts data (OCR + intelligent parsing)
- Step 3: Select form template, auto-fill, download
- Technical advantage: Open-source OCR = no per-page costs

SLIDE 6: MARKET OPPORTUNITY
- UAE has 50,000+ business setup transactions/year
- 500+ PRO agencies handling paperwork
- Each agency processes 50-200 clients/month
- Target addressable market: $10M+ annually
- Expansion potential: GCC region, then global

SLIDE 7: COMPETITIVE ADVANTAGE
| Feature | IntelliFill | Manual Process | Generic Tools |
|---------|-------------|----------------|---------------|
| Processing time | 90 seconds | 45 minutes | 15 minutes |
| Accuracy | 93%+ | Error-prone | Varies |
| Cost per page | Free (OSS OCR) | Labor cost | $0.01-0.03/page |
| UAE form templates | Pre-built | N/A | None |

SLIDE 8: BUSINESS MODEL
- Pricing tiers:
  - Starter: $99/month (solo agents, 100 docs)
  - Professional: $299/month (small agencies, 500 docs)
  - Agency: $599/month (unlimited, team features)
- Unit economics: High margin due to open-source OCR
- LTV potential: PRO agencies are sticky (ongoing need)

SLIDE 9: TRACTION & ROADMAP
- Current: MVP complete, 70% production ready
- Q1 2025: 5 pilot customers, $500+ MRR
- Q2 2025: Product-led growth launch, $5K MRR
- Q3 2025: Team features, Arabic UI, $15K MRR
- Q4 2025: GCC expansion, $50K MRR

SLIDE 10: THE ASK
- Seeking: First 5 founding customers
- Offer: 50% discount for early adopters
- In return: Feedback, testimonial rights
- Contact: [Your email/calendar link]

**Design Style:**
- Clean, professional, modern
- Primary color: Blue (#2563eb)
- Secondary: Slate gray
- White backgrounds with minimal text
- Use icons and visuals over text where possible
- Sans-serif font (Inter or similar)

Generate this as a detailed slide-by-slide outline with suggested visuals and speaker notes for each slide.
```

## Prompt 2: Demo Video Script (90 seconds)

Copy this entire prompt into Gemini:

```
Write a professional 90-second demo video script for IntelliFill, an AI-powered form automation tool for UAE PRO agencies.

**Video Details:**
- Duration: 90 seconds
- Style: Professional but approachable, not overly corporate
- Audience: UAE PRO agents and agency owners
- Goal: Show the product in action, demonstrate time savings

**Product Features to Show:**
1. Document upload (drag and drop)
2. AI data extraction (showing extracted fields)
3. Form template selection
4. One-click auto-fill
5. Download completed form

**Key Messages:**
- "From 45 minutes to 90 seconds"
- "93% accuracy - get it right the first time"
- "All your client data in one place"
- "No more copy-pasting"

**Script Structure:**

[0:00-0:10] HOOK
- Open with the pain point
- Something like: "How many hours did you spend filling forms this week?"

[0:10-0:25] PROBLEM
- Show the manual process (visual of someone typing, copy-pasting)
- Stats: "PRO agents spend 4 hours every day on manual data entry"
- Emphasize the tedium and error potential

[0:25-0:55] SOLUTION DEMO
- "There's a better way. Meet IntelliFill."
- Show the actual product:
  - Upload a passport (5 seconds)
  - AI extraction happening (5 seconds)
  - Review extracted data (5 seconds)
  - Select form, click fill (5 seconds)
  - Download completed form (5 seconds)
  - Show the timer: "Total time: 90 seconds"

[0:55-1:15] BENEFITS
- "93% accuracy means fewer errors and rejections"
- "One client profile, unlimited forms"
- "Works with Emirates ID, passports, trade licenses"
- "No more switching between windows"

[1:15-1:25] SOCIAL PROOF (if available)
- Quote from beta user or design partner
- Or: "Trusted by UAE PRO professionals"

[1:25-1:30] CALL TO ACTION
- "Try IntelliFill free for 14 days"
- "Visit intellifill.com"
- Logo and website

**Tone:**
- Confident but not arrogant
- Empathetic to the user's pain
- Professional, suitable for B2B

**Visual Suggestions:**
- Clean screen recordings of the actual product
- Split screen showing manual vs. automated process
- Timer graphic showing 90 seconds vs 45 minutes
- Checkmark animations when form is filled

Generate:
1. Complete script with timestamps
2. Visual descriptions for each scene
3. Suggested background music style
4. On-screen text/graphics for each section
```

## Prompt 4: Customer Interview Questions Script

```
Create a customer discovery interview script for IntelliFill targeting UAE PRO agencies.

**Context:**
- We're validating product-market fit for an AI form automation tool
- Target interviewee: PRO agents or agency owners
- Goal: Understand their workflow, pain points, and willingness to pay
- Duration: 15-20 minutes

**Interview Structure:**

INTRO (2 minutes)
- Thank them for their time
- Explain this is research, not a sales call
- Ask permission to take notes

CURRENT WORKFLOW (5 minutes)
- Open-ended questions about their daily work
- Specific questions about form filling process

PAIN POINTS (5 minutes)
- What frustrates them most?
- What have they tried to solve this?
- How much time/money does it cost them?

SOLUTION VALIDATION (5 minutes)
- Describe the product concept
- Get their reaction
- Would they use it? Would they pay?

WRAP-UP (3 minutes)
- Ask if they know others who'd be interested
- Thank them, next steps

Generate:
1. Full script with exact questions
2. Probing follow-up questions for each section
3. Warning signs that indicate no product-market fit
4. Positive signals that indicate strong interest
```

1. Copy the prompt you need

4. For the pitch deck: Use Google Slides or your preferred tool to create visuals

**Tips:**
- Feed Gemini screenshots of your actual product for more accurate descriptions
- Ask Gemini to iterate on specific slides or sections if needed
- Use Gemini's image generation for slide visuals if available
