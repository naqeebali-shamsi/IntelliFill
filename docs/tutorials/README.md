# Tutorials

Tutorials are **learning-oriented** lessons that take the reader through a series of steps to complete a project. They focus on learning by doing.

---

## Available Tutorials

### [Getting Started](./getting-started.md)

**Time**: 30 minutes

Set up your development environment from scratch:
- Install prerequisites (Node.js, Bun, Docker)
- Clone the repository
- Configure environment variables
- Start development servers
- Verify everything works

### [Your First Document](./first-document.md)

**Time**: 15 minutes

Upload and process your first document:
- Log in to the application
- Upload a sample document
- Watch OCR extraction in action
- View extracted data
- Fill a PDF form

### [Understanding the Workflow](./understanding-workflow.md)

**Time**: 20 minutes

Learn how IntelliFill processes documents:
- Document upload flow
- OCR extraction process
- Data extraction and validation
- Field mapping
- PDF form filling

---

## Tutorial Philosophy

Tutorials in this section follow these principles:

1. **Learn by doing** - Each tutorial has hands-on exercises
2. **Achievable goals** - Clear, completable objectives
3. **Immediate feedback** - See results quickly
4. **Minimal explanation** - Focus on steps, not concepts
5. **Safe environment** - Can't break anything

---

## Prerequisites

Before starting any tutorial, ensure you have:

- **Operating System**: Windows 10/11, macOS, or Linux
- **Git**: For cloning the repository
- **Text Editor**: VS Code recommended with Cursor extension
- **Terminal**: PowerShell (Windows), Terminal (macOS/Linux)
