## Slide 5: How It Works

**Headline**: Intelligent Technology Stack

**Architecture Diagram (Simplified)**:

```
┌──────────────────────────────────────────────────────┐
│                    IntelliFill                        │
│                                                       │
│  ┌─────────┐   ┌─────────────┐   ┌─────────────┐    │
│  │ Document │   │ Extraction  │   │   Profile   │    │

│                                                       │
│  ┌─────────┐   ┌─────────────┐   ┌─────────────┐    │
│  │ Field   │   │    Form     │   │   Output    │    │
│  │ Mapper  │──►│   Filler    │──►│  Generator  │    │

│       │              │                  │            │
│       ▼              ▼                  ▼            │
│  Levenshtein   PDF Text/Checkbox   Download/Export   │
│  Matching        Filling            PDF/JSON/CSV     │
└──────────────────────────────────────────────────────┘
```

**Key Technical Points**:
- Open-source OCR = **$0 per-document cost**
- Levenshtein algorithm = **Automatic field matching**
- Profile memory = **Data entered once, used forever**
